# MML Book — Mathematics for Machine Learning

**Authors:** Marc Peter Deisenroth, A. Aldo Faisal, Cheng Soon Ong  
**Publisher:** Cambridge University Press (freely available at mml-book.github.io)  
**Role:** Primary mathematical reference for M.Tech Semester 1  
**Current chapter:** Ch.1 Introduction & Notation (finishing June 22); Ch.2 Linear Algebra starts June 23

---

## Overview
The MML book builds the mathematical foundations needed to read ML research and implement algorithms: linear algebra, analytic geometry, matrix decompositions, vector calculus, probability — then Part II applies all of it to classical ML algorithms. It is not a cookbook. Reading it requires working through exercises. If you're treating it as a reading-only source, you're doing it wrong.

## Key Notation (Ch.1 Reference)

| Symbol | Meaning |
|--------|---------|
| *a*, *x*, *λ* | Scalars (lowercase italic) |
| **x**, **w**, **b** | Vectors (lowercase bold) |
| **A**, **W**, **Σ** | Matrices (uppercase bold) |
| 𝒳, 𝒴 | Sets (calligraphic) |
| ℝⁿ | n-dimensional real vector space |
| ∈ | "is an element of" |
| := | "is defined as" |
| ∝ | "is proportional to" |
